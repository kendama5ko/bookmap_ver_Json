### 本の追加方法
①管理ボタンを押す  
②タイトル、著者、ジャンル、ページ数を入力  
③追加ボタンを押す[^1]  
④右上のリストから追加した本を選択  
⑤読んだページ数を入力  
⑥記録する  
[^1]: 本の追加には必ずページ数を入力してください。
